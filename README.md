# **Tela de Login - Projeto Login.** link do site [https://thiagobatista5.github.io/interface/]

## Descrição

O projeto **Tela de Login** tem como objetivo fornecer uma interface de login para membros de uma plataforma, permitindo o acesso de sócios por meio de um formulário de login simples. O projeto é desenvolvido utilizando HTML, CSS, JS com estrutura de design responsiva e clean.

A página de login contém campos para CPF e senha, além de um link para o cadastro de novos membros. A interface é projetada para ser simples, funcional e intuitiva, permitindo fácil navegação e interação com os usuários.

---

## Funcionalidades

* **Campo de CPF:** O usuário pode inserir seu CPF para identificar-se como sócio da plataforma.
* **Campo de Senha:** A senha associada ao CPF será utilizada para autenticar o acesso do usuário.
* **Design Responsivo:** A página de login se adapta a diferentes tamanhos de tela, proporcionando uma experiência de uso agradável tanto em dispositivos móveis quanto em desktops.
